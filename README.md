# СТО Level - Официальный сайт автосервиса

Современный адаптивный веб-сайт для автосервиса СТО Level с красно-черной цветовой схемой.

## 🚗 Функциональность

### 1. Главная страница
- Яркий промо-дизайн с hero-секцией
- Информация о компании
- Уникальные предложения
- Контактные данные
- Призыв к действию

### 2. Услуги
Разделены на две категории с табами:

**Ремонт авто:**
- Замена колодок (от 3,500₽)
- Ремонт подвески (от 8,500₽)
- Диагностика двигателя (от 2,500₽)
- Замена масла (от 1,800₽)
- Шиномонтаж (от 2,000₽)
- Обслуживание кондиционера (от 3,000₽)

**Тюнинг:**
- **Stage 1** - Базовый тюнинг (от 35,000₽)
- **Stage 2** - Умеренный тюнинг (от 75,000₽)
- **Stage 3** - Максимальный тюнинг (от 150,000₽)

### 3. Галерея работ
- Фотографии автомобилей после тюнинга
- Подписи с моделью и выполненными работами
- Интерактивные превью с эффектами hover

### 4. Корзина услуг
- Добавление услуг в корзину
- Отображение цены и итоговой суммы
- Возможность удаления услуг
- Оформление заказа

### 5. Личный кабинет
**Регистрация включает:**
- ФИО
- Email
- Пароль
- Информация об автомобиле (марка, модель, год)

**Личный кабинет содержит:**
- Личные данные пользователя
- Информация об автомобиле
- История оказанных услуг

### 6. Дополнительные возможности
- Форма обратной связи
- Раздел с отзывами клиентов
- Возможность оставить отзыв
- Интеграция с социальными сетями (VK, Telegram, Instagram, YouTube)

## 🎨 Дизайн

- **Цветовая схема:** Красно-черная
- **Адаптивность:** Полностью адаптивный дизайн для всех устройств
- **Технологии:**
  - HTML5
  - CSS3 (с переменными и современными анимациями)
  - Vanilla JavaScript (без зависимостей)
  - Font Awesome для иконок

## 🚀 Запуск проекта

1. Просто откройте `index.html` в браузере
2. Все данные хранятся в localStorage браузера
3. Никаких дополнительных зависимостей или серверов не требуется

## 💾 Хранение данных

Сайт использует localStorage для хранения:
- Корзины покупок
- Данных пользователей
- Текущей сессии
- Истории услуг

## 🗄️ База данных

Для демонстрации структуры базы данных откройте файл `database-demo.html` в браузере.

### Структура таблиц:

1. **users** - Зарегистрированные пользователи
2. **services** - Каталог услуг
3. **service_history** - История оказанных услуг
4. **reviews** - Отзывы клиентов
5. **contact_requests** - Заявки с формы обратной связи

Полный SQL-скрипт для создания базы данных находится в файле `database-schema.sql`
