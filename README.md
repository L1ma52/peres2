## 🚀 Запуск проекта

### ⚡ Быстрый старт (5 минут)

**Смотрите файл `QUICK_START.md` - там все просто и пошагово!**

### Frontend (только HTML/CSS/JS)
1. Просто откройте `index.html` в браузере
2. Все данные будут храниться в localStorage браузера

### Backend (с подключением к БД)

#### 1. Установите зависимости:
```bash
npm install
```

#### 2. Настройте PostgreSQL и создайте базу данных:

**См. подробную инструкцию в `DATABASE_SETUP.md`**

Краткая версия:
```bash
# Создайте БД
createdb -U postgres stolevel

# Импортируйте схему
psql -U postgres -d stolevel -f database-schema.sql
```

**Альтернатива:** Используйте pgAdmin 4 (графический интерфейс) - это проще!

#### 3. Создайте файл `.env` с настройками:
```bash
cp .env.example .env
```

Отредактируйте `.env`:
```env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=stolevel
DB_USER=postgres
DB_PASSWORD=ваш_пароль_postgres
PORT=3000
JWT_SECRET=очень_секретный_ключ_123
NODE_ENV=development
```

#### 4. Запустите сервер:
```bash
npm start
```

#### 5. Откройте браузер:
```
http://localhost:3000
```

Фронтенд автоматически подключится к API!

**Проверьте работу:** http://localhost:3000/api/health

## 🗄️ База данных

Для демонстрации структуры базы данных откройте файл `database-demo.html` в браузере.

### Структура таблиц:

1. **users** - Зарегистрированные пользователи
2. **services** - Каталог услуг
3. **service_history** - История оказанных услуг
4. **reviews** - Отзывы клиентов
5. **contact_requests** - Заявки с формы обратной связи

Полный SQL-скрипт для создания базы данных находится в файле `database-schema.sql`

## 🔌 API Endpoints

### Пользователи
- `POST /api/auth/register` - Регистрация пользователя
- `POST /api/auth/login` - Вход в систему
- `GET /api/auth/profile` - Получить профиль пользователя
- `PUT /api/auth/profile` - Обновить профиль

### Услуги
- `GET /api/services` - Получить список услуг
- `GET /api/services/:id` - Получить конкретную услугу

### История услуг
- `GET /api/service-history` - Получить историю услуг пользователя
- `POST /api/service-history` - Добавить запись в историю

### Отзывы
- `GET /api/reviews` - Получить все отзывы
- `POST /api/reviews` - Добавить отзыв

### Контактные запросы
- `POST /api/contact` - Отправить сообщение через форму обратной связи
