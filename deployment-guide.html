1
Подготовка GitHub

Загрузите ваш проект на GitHub:

git init git add . git commit -m "Initial commit" git remote add origin https://github.com/ваш-username/sto-level.git git push -u origin main

Если у вас нет GitHub аккаунта - создайте на github.com

2
База данных на Render
  1. Зарегистрируйтесь на Render.com
  2. New + → PostgreSQL
  3. Name: stolevel-db
  4. Plan: Free
  5. Create Database
⚠️ Важно! Скопируйте Internal Database URL - он понадобится!

Импортируйте схему БД:

# Скопируйте PSQL Command из Render # Затем: psql ваша_команда_подключения < database-schema.sql
✅ База данных готова!
3
Backend на Render
  1. New + → Web Service
  2. Подключите GitHub репозиторий
  3. Name: stolevel-api
  4. Build Command: npm install
  5. Start Command: node backend/server.js
  6. Plan: Free

Добавьте Environment Variables:

DATABASE_URL=ваш_internal_database_url_из_шага_2 PORT=3000 JWT_SECRET=ваш_секретный_ключ_123456 NODE_ENV=production

После деплоя проверьте: https://ваш-url.onrender.com/api/health

4
Frontend на Vercel

Создайте файл vercel.json:

{ "version": 2, "builds": [{"src": "public/**", "use": "@vercel/static"}], "routes": [{"src": "/(.*)", "dest": "/public/$1"}] }

Обновите API_URL в public/script.js:

const API_URL = 'https://stolevel-api.onrender.com/api';
  1. Зарегистрируйтесь на Vercel.com
  2. Add New → Project
  3. Import ваш GitHub репозиторий
  4. Output Directory: public
  5. Deploy
🎉 Готово! Ваш сайт в интернете: https://sto-level.vercel.app
🚂 Railway - проще всего!
Всё в одном месте, настраивается за 15 минут.
1
Регистрация на Railway
  1. Перейдите на Railway.app
  2. Войдите через GitHub
  3. Добавьте карту для $5 кредита
2
Создание проекта
  1. New Project → Deploy from GitHub repo
  2. Выберите репозиторий sto-level
  3. New → Database → Add PostgreSQL

Railway автоматически создаст БД и добавит DATABASE_URL!

3
Настройка сервиса

В вашем backend сервисе:

  • Settings → Start Command: node backend/server.js
  • Variables → Add:
    PORT=3000 JWT_SECRET=ваш_секретный_ключ NODE_ENV=production
4
Импорт данных
  1. PostgreSQL сервис → Data → Query
  2. Вставьте содержимое database-schema.sql
  3. Execute
✅ База данных заполнена!
5
Frontend на Vercel

Точно так же как в бесплатном варианте (Шаг 4).

Обновите API_URL на ваш Railway домен.

🎉 Готово! Сайт работает и не засыпает!
