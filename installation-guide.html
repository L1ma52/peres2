🚗 Гид по установке СТО Level

Пошаговая инструкция для создания базы данных PostgreSQL

🖥️ pgAdmin (Просто)
🪟 Windows
🍎 macOS
🐧 Linux
⭐ РЕКОМЕНДУЕТСЯ ДЛЯ НОВИЧКОВ!
pgAdmin - это графический интерфейс для PostgreSQL. Не нужно знать команды!
1
Установите PostgreSQL
Windows: 📥 Скачать PostgreSQL для Windows
  • Запустите установщик
  • ⚠️ ВАЖНО: Запомните пароль для пользователя postgres!
  • Оставьте порт по умолчанию: 5432
  • Убедитесь, что pgAdmin 4 тоже установлен (галочка в установщике)
2
Запустите pgAdmin 4
  • Windows: Пуск → pgAdmin 4
  • macOS: Applications → pgAdmin 4
  • При первом запуске придумайте master password (любой)
📸 Откроется веб-интерфейс pgAdmin в браузере
3
Подключитесь к серверу
  • В левой панели раскройте Servers
  • Кликните на PostgreSQL 15 (или другую версию)
  • Введите пароль, который создали при установке
  • Поставьте галочку "Save password"
4
Создайте базу данных
  • Правый клик на "Databases"
  • Выберите Create → Database...
  • В поле "Database" введите: stolevel
  • Owner оставьте: postgres
  • Нажмите Save
✅ База данных stolevel создана!
5
Импортируйте таблицы
  • Кликните на базу данных stolevel
  • В верхнем меню: Tools → Query Tool
  • Нажмите 📁 File → Open
  • Найдите и откройте файл database-schema.sql из вашего проекта
  • Нажмите ▶ Execute/Refresh или клавишу F5
✅ Таблицы созданы и заполнены тестовыми данными!
6
Проверьте результат
В левой панели раскройте:
stolevel → Schemas → public → Tables
Вы должны увидеть 5 таблиц:
  • ✅ contact_requests
  • ✅ reviews
  • ✅ service_history
  • ✅ services
  • ✅ users
7
Настройте проект
Откройте терминал в папке проекта и выполните:
# Скопируйте пример конфигурации
cp .env.example .env
Откройте файл .env в блокноте и укажите ваш пароль:
DB_PASSWORD=ваш_пароль_postgres
8
Запустите сервер
npm install
npm start
🎉 Готово! Откройте браузер: http://localhost:3000
1
Откройте PowerShell или CMD
Нажмите Win + R, введите cmd или powershell
2
Перейдите в папку проекта
cd C:\путь\к\вашему\проекту\sto-level
3
Создайте базу данных
createdb -U postgres stolevel
# Введите пароль postgres когда попросит
Если команда не найдена, используйте полный путь:
"C:\Program Files\PostgreSQL\15\bin\createdb.exe" -U postgres stolevel
4
Импортируйте схему
psql -U postgres -d stolevel -f database-schema.sql
# Введите пароль postgres
✅ База данных готова!
